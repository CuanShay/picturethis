Command Prompt

  Visualiser

Type at any stage, to get a list of available commands and definitions. alternatively you can exit the simulator by entering the command.
To change directory, we use the command of the command line terminal. In this case we navigate up a directory by typing and note the change in the directory tree. To navigate down into a directory, in this case documents type "CD" followed by the directory name. The target location must be in your path.

To list the current directory contents type the command or additionally we can add a directory name that is in our path to see its contents.

Note that our location remains unchanged and the documents directory here contains one file.
To clear the display type
followed by enter. This command clears the display aesthetically.

The command prints out text to the display. Type and the terminal will output "HELLO". It can also output the resulting text to a text file. Enter and see the result by listing the current directory.

Similarly we can make a directory or folder using the command followed by the name we wish to give the folder.

To rename a folder. Type followed by the folder to be renamed and the new folder name. and see the result in the file tree.

To delete a folder and all of its contents, enter followed by the name of the folder we wish to delete.

In this case, we deleted the documents directory.
